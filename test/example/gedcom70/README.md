# Example FamilySearch GEDCOM 7.0 Files

from the [Tools for FamilySearch GEDCOM](https://gedcom.io/tools/#example-familysearch-gedcom-70-files) page on https://gedcom.io

| File                                                 | Notes                                                                                                                                                                                                                                                                                                                                                  |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [age.ged](./age.ged)                                 | This file contains various test cases for AGE payloads.                                                                                                                                                                                                                                                                                                |
| [escapes.ged](./escapes.ged)                         | This file contains @ characters that are doubled only when they appear at the start of a line string.                                                                                                                                                                                                                                                  |
| [extension-record.ged](./extension-record.ged)       | This file contains a \_LOC record, which is referenced from an individual with a \_LOC tag.                                                                                                                                                                                                                                                            |
| [extensions.ged](./extensions.ged)                   | This file contains GEDCOM extensions of various forms.                                                                                                                                                                                                                                                                                                 |
| [lang.ged](./lang.ged)                               | This file contains many different LANG payloads.                                                                                                                                                                                                                                                                                                       |
| [filename-1.ged](./filename-1.ged)                   | This file contains many different FILE payloads.                                                                                                                                                                                                                                                                                                       |
| [long-url.ged](./long-url.ged)                       | This file contains a URL of a submitter that is very long, to test the ability to parse very long lines.                                                                                                                                                                                                                                               |
| [maximal70.ged](./maximal70.ged)                     | This file attempts to exercise all standard tags in various locations, and all standard enumeration values.                                                                                                                                                                                                                                            |
| [maximal70.gdz](./maximal70.gdz)                     | This FamilySearch GEDZIP 7.0 file attempts to exercise all standard tags in various locations, and all standard enumeration values, and also includes a minimal local audio file.                                                                                                                                                                      |
| [maximal70-lds.ged](./maximal70-lds.ged)             | This file includes all of maximal70-tree1.ged plus the following structures: BAPL, CONL, <<LDS_ORDINANCE_DETAIL>>.DATE, ENDL, INIL, SLGC, SLGS, and TEMP.                                                                                                                                                                                              |
| [maximal70-memories1.ged](./maximal70-memories1.ged) | This file includes all of maximal70-tree1.ged plus the following structures: INDI.OBJE, OBJE (i.e., <<MULTIMEDIA_RECORD>>), OBJE.FILE, and OBJE.FILE.FORM.                                                                                                                                                                                             |
| [maximal70-memories2.ged](./maximal70-memories2.ged) | This file includes all of maximal70-memories1.ged plus the following structures: FAM.OBJE, <<EVENT_DETAIL>>.OBJE, and SOUR.OBJE.                                                                                                                                                                                                                       |
| [maximal70-tree1.ged](./maximal70-tree1.ged)         | This file includes all of minimal70.ged plus the following structures: ABBR, AUTH, BIRT, BURI, CHIL, CHR, <<EVENT_DETAIL>>.DATE, DEAT, FAM (<<FAMILY_RECORD>>), FAM.SOUR, INDI.FAMC, FAMS, FAM.HUSB, INDI (<<INDIVIDUAL_RECORD>>), INDI.SOUR, MARR, INDI.NAME, INDI.NOTE, PAGE, <<EVENT_DETAIL>>.PLAC, PUBL, SEX, SOUR (<<SOURCE_RECORD>>), SOUR.TITL. |
| [maximal70-tree2.ged](./maximal70-tree2.ged)         | This file includes all of maximal70-tree1.ged plus many more <<INDIVIDUAL_ATTRIBUTE_STRUCTURE>>, <<INDIVIDUAL_EVENT_STRUCTURE>>, <<FAMILY_ATTRIBUTE_STRUCTURE>>, and <<FAMILY_EVENT_STRUCTURE>> structures, as well as <<EVENT_DETAIL>>.SOUR, INDI.NAME.SOUR, NOTE.SOUR, and <<EVENT_DETAIL>>.NOTE.                                                    |
| [minimal70.ged](./minimal70.ged)                     | This file is the smallest legal FamilySearch GEDCOM 7.0 file.                                                                                                                                                                                                                                                                                          |
| [minimal70.gdz](./minimal70.gdz)                     | This file is the smallest legal FamilySearch GEDZIP 7.0 file.                                                                                                                                                                                                                                                                                          |
| [notes-1.ged](./notes-1.ged)                         | This file contains many different uses of NOTE and SNOTE.                                                                                                                                                                                                                                                                                              |
| [obje-1.ged](./obje-1.ged)                           | This file contains many different uses of OBJE records.                                                                                                                                                                                                                                                                                                |
| [remarriage1.ged](./remarriage1.ged)                 | This file contains a couple who married, divorced, and then later remarried, stored as one family.                                                                                                                                                                                                                                                     |
| [remarriage2.ged](./remarriage2.ged)                 | This file contains a couple who married, divorced, and then later remarried, stored as two separate families.                                                                                                                                                                                                                                          |
| [same-sex-marriage.ged](./same-sex-marriage.ged)     | This file contains a same sex marriage.                                                                                                                                                                                                                                                                                                                |
| [voidptr.ged](./voidptr.ged)                         | This file contains several @VOID@ references.                                                                                                                                                                                                                                                                                                          |
| [xref.ged](./xref.ged)                               | This file contains a number of different cross-reference identifier formats.                                                                                                                                                                                                                                                                           |
